# OrangeHRM QA Portfolio

## Project Overview

This repository contains my QA portfolio project created using the OrangeHRM demo application.

The purpose of this project is to demonstrate:

* Manual Testing
* Test Case Design
* Bug Reporting
* Test Documentation
* Playwright Test Automation

## Application Under Test

OrangeHRM Demo Application

Modules covered:

* Login
* Dashboard
* Employee Management
* Search Functionality
* User Management
* Logout
